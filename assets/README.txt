Holiday Panic Factory - Asset Directory
==================================================

Place your game assets in the appropriate folders:

backgrounds/  - Background images (1280x720 PNG)
characters/   - Character sprites (elf head, etc.)
toys/         - Toy sprites (good/bad variants)
wrapping/     - Wrapping paper sprites
bows/         - Bow/ribbon sprites
sounds/music/ - Background music (.ogg files)
sounds/sfx/   - Sound effects (.wav files)
ui/           - UI elements and icons

See ../ASSET_ORGANIZATION.md for detailed requirements.
